# Citing Amigo

If you use Amigo in your research or publications, please cite our work. Citations help us justify continued development and support for the project.

## BibTeX Entry

```bibtex
@software{amigo,
  author = {Kennedy, Graeme J.},
  title = {Amigo: A Friendly Library for MDO on HPC},
  year = {2024},
  publisher = {GitHub},
  url = {https://github.com/your-org/amigo},
  version = {0.0.1}
}
```

## Related Publications

If you use specific features of Amigo, please also consider citing the relevant publications:

### Automatic Differentiation (A2D)

If you use Amigo's automatic differentiation capabilities:

```bibtex
@article{a2d2023,
  author = {Author Name},
  title = {A2D: Automatic Differentiation for Structural Optimization},
  journal = {Journal Name},
  year = {2023},
  volume = {XX},
  pages = {XXX-XXX}
}
```

### Multidisciplinary Design Optimization

For MDO applications:

```bibtex
@article{mdo2024,
  author = {Kennedy, Graeme J.},
  title = {Scalable Multidisciplinary Design Optimization on HPC Systems},
  journal = {Structural and Multidisciplinary Optimization},
  year = {2024},
  volume = {XX},
  pages = {XXX-XXX}
}
```

Share your publications, projects, or applications built with Amigo. We love to see what the community creates!
